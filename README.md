**Exit** is a single-player 3D adventure game developed in **Unity 3D**. The player’s objective is to escape a castle by exploring the environment, collecting scattered keys, and battling enemies to unlock the main gate. The game combines exploration, combat, and puzzle-solving elements to create an engaging gameplay experience.

This project was developed by a **7-member team** following **Agile and Scrum methodologies**, with contributions across multiple areas including map design, character development, animations, physics, and sound design.
